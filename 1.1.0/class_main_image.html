◆ mouseReleaseEvent()

void MainImage::mouseReleaseEvent ( QMouseEvent * event)
overrideprotected

Handle mouse move events.

Parameters
eventPointer to the mouse event

This function is called when the mouse is moved over the widget. It updates the hover background color and style sheet.

It also handles the cropping rectangle drawing if the mouse is pressed and the cropping mode is enabled.

◆ paintEvent()

void MainImage::paintEvent ( QPaintEvent * event)
overrideprotected

Paint event handler for the MainImage widget.

Parameters
eventPointer to the paint event

This function is called when the widget needs to be repainted. It draws the cropping rectangle.

It also draws detected objects if the personsEditor is enabled and there are detected objects in the image data.
