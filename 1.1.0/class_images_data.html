◆ setImageNumber()

void ImagesData::setImageNumber ( int nbr)

Set the image number to the given value. If the value is out of range, it will be clamped to the valid range.

Parameters
nbrThe new image number to set.

This function sets the image number to the specified value. If the value is less than 0, it will be set to 0. If it exceeds the size of currentImagesData, it will be set to the last index of currentImagesData.
